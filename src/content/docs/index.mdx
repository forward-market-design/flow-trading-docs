## About

This site hosts high-level user documentation for the reference flow trading implementation provided by [Forward Market Design](https://forwardmarketdesign.com).
In addition to quick start guides, there are explanations of the bidding language on how a bidder can concretely leverage these features to achieve desired outcomes.
